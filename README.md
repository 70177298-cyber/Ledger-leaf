# Ledger & Leaf

A simple expense tracker built with HTML, CSS, and JavaScript.

## Features
- Add, view, and delete expenses
- Filter expenses by category
- Category spending chart
- 7-day spending trend chart
- Data stored locally in your browser
- Responsive design for mobile and desktop

## Categories
- Food
- Transport
- Entertainment
- Utilities
- Health
- Education
- Other

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- LocalStorage

Privacy

All expense data is stored in your browser using LocalStorage. No data is uploaded or shared.
